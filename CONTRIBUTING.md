# Contributing to Connect Platform

First off, thank you for considering contributing to Connect! It's people like you that make Connect such a great tool.

## How Can I Contribute?

### Reporting Bugs
*   Check the [Issues](https://github.com/Utility-sys/Connect-Platform/issues) to see if the bug has already been reported.
*   If you can't find an open issue that describes the problem, [open a new one](https://github.com/Utility-sys/Connect-Platform/issues/new). Include a clear title and a detailed description.

### Suggesting Enhancements
*   [Open an issue](https://github.com/Utility-sys/Connect-Platform/issues/new) with a clear title and a description of the proposed enhancement.

### Pull Requests
1.  Fork the repo and create your branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  Ensure the test suite passes.
4.  Issue that pull request!

## Styleguide
*   Use camelCase for variables and functions.
*   Use PascalCase for React components.
*   Write clear, descriptive commit messages.
